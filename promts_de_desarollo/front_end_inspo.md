## Prompt de diseño — Promptary UI

**Design a modern minimalist SaaS web application called "Promptary", a premium prompt library platform for data scientists and AI engineers.**

The interface should feel like a combination of **Linear, Notion, Vercel, and modern AI tools**: clean, elegant, dark mode, highly organized, with strong visual hierarchy.

### Overall Style

* Minimalistic dark interface
* Premium developer tool aesthetic
* Clean spacing
* Rounded cards
* Subtle borders
* Soft shadows
* Modern typography
* Professional but creative
* Avoid excessive colors or gradients

### Color Palette

Primary background:

* Almost black / charcoal
* #0D0F12

Cards:

* Dark gray
* #15181D

Borders:

* Soft gray transparency
* #262B33

Accent:

* Electric violet / AI purple
* #8B5CF6

Secondary accent:

* Soft blue
* #3B82F6

Success:

* Green for metrics
* #22C55E
