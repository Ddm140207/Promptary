Promptary — AI Prompt Knowledge Library for Data Scientists
Concepto

Promptary es una plataforma colaborativa diseñada para que científicos de datos, desarrolladores e investigadores puedan organizar, compartir y mejorar prompts utilizados en flujos de trabajo con inteligencia artificial.

La idea nace de un problema común en equipos que trabajan con IA: los mejores prompts suelen perderse entre conversaciones, documentos o proyectos individuales. Promptary busca convertirse en una biblioteca centralizada donde la comunidad pueda descubrir, reutilizar y optimizar prompts especializados para diferentes áreas de ciencia de datos.

En lugar de crear prompts desde cero cada vez, los usuarios pueden acceder a una colección de conocimiento compartido construida por la comunidad.

Objetivo

Crear un repositorio vivo de prompts de alta calidad enfocados en:

📊 Exploratory Data Analysis (EDA)
🧹 Data Cleaning & Preparation
🤖 Machine Learning
📈 Data Visualization
📝 Technical Documentation
🐛 Debugging & Code Optimization
🗄️ SQL & Database Analysis
🧠 AI Workflow Optimization
Comunidad Open Source

Promptary funcionará bajo una filosofía colaborativa donde científicos de datos y desarrolladores de todo el mundo puedan contribuir con sus propios prompts.

Cada contribución mantiene información del creador:

Esto permite que cada prompt tenga contexto, autoría y trazabilidad, convirtiendo la plataforma en una comunidad de conocimiento compartido.
